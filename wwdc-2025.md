## [Platforms State of the Union](https://developer.apple.com/videos/play/wwdc2025/102)
- Liquid Glass. Dynamic island 이후로 top/bottom navigation bar를 어떻게 해야할지 결심이 선 느낌이다. Liquid Glass와도 잘 맞는 것 같다. 일관되게 모든 모서리도 다 채우고.
- On-device Apple Intelligence. Text extraction, summarization, etc. Response도 batch/streaming generation을 선택할 수 있음. `@Generable`로 모델을 정의하면, 해당 타입을 response로 생성해주기도.
- Playground의 non-UI preview. Prompt engineering과 잘 맞는다.
- Xcode 26. Integrate ChatGPT to Xcode. 디자인 이미지를 drag&drop 하면 View code 작성.
- Conversation history 제공. 원하는 시점으로 돌아가서 다른 시도 가능. 다른 AI tool들도 제공할 법한데, 찾아볼 생각을 안했네.
- Anthropic API key를 넣으면 provider를 변경하는 것도 가능.
- Swift 6.2. `@MainActor`가 디폴트가 되고, 백그라운드로 돌리고 싶으면 `@concurrent`?
- WebKit. Added a new declarative `WebView` for SwiftUI.
- Idle prefetch. Frame 간 idle time에 다음 frame 연산을 땡겨서 하는 SwiftUI feature. 다른 platform은 원래 되고 있었고, 이번에 mac을 지원한다고.
